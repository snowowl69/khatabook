# KhataBook

KhataBook is a desktop billing and stock management app for small shopkeepers, built with Electron, React, and SQLite.

## Features

- Billing and invoice management
- Customer and supplier records
- Inventory and stock tracking
- Purchase and expense workflows
- Cash register, day book, and reports

## Tech Stack

- Electron
- Electron Vite
- React
- TypeScript
- SQLite via better-sqlite3

## Getting Started

Install dependencies:

```bash
npm install
```

Run the app in development:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
